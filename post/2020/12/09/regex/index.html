REGEX

December 9, 2020    R Markdown plot regression

When first introduced to Regular Expressions (RegEx), I found myself extremely confused. How could a seemingly random group of characters translate into a search pattern? Upon further investigation, filled with boughts of frustration, I found myself fascinated. RegEx is essentially shortahdn notation used to specify search criteria, and its practical applications are present throughout our everyday, technological uses.

For example, when creating a login, some websites require passwords to fulfill a set of criteria denoting length, special characters, etc. RegEx can be used to do this! Although I am still an amateur when it comes to Regular Expressions and can't thoroughly explain the typical RegEx code used to enforce password strength parameters, I did find this application interesting and highly relatable!

As a biology major, I was excited to learn that RegEx can be applied to nucleotide sequences! For example, when given restriction enzyme binding sites, ANTAAT and GCRWTG, the function re.findall() can be utilized to search for these sites in sequence.
